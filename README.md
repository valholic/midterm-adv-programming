# Midterm Advanced Programming

The goal of this code is to find a number inside an array as soon as we can using binary search. each problem containing:
1. A sorted list array of integers
2. An integer target to search for

Example: 
Problem 1.
array = [1, 3, 5, 7, 9]
target = 7
output = 4

Problem 2.
array = [-1, 0, 2, 3, 4, 5, 6, 9]
target = 9
output = 7

The constraints you would face is :
- The data must be sorted
- if the array has duplicate elements, it might cause some bugs
- the array maybe empty
- integer limit
